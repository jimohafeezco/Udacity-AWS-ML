

# Get started
```
git clone https://github.com/MrDaubinet/Dog-Breed-Classifier
cd Dog-Breed-Classifier
jupyter notebook
```
# Requirements
Jupyter notebook must be installed.
Python must be installed. The following python modules must be installed.
```
opencv-python
jupyter
matplotlib
pandas
numpy
pillow
scipy
tqdm
scikit-learn
scikit-image
seaborn
h5py
ipykernel
bokeh
pickleshare
```
